     (ListboxWidget: in Tk_ListBox; First: in String; Last: in String := "")
      return String with
      Pre => First /= "";

PARAMETERS

 ListboxWidget - Tk_ListBox which will be queried for elements

RESULT

 If both are present, return list of elements from First to Last. If
